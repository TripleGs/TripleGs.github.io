ImagineTTRPGs Website

Accessable At: https://imaginettrpgwebsite.web.app

Project Details

This was a project that I created as a way to explore what I could do using my coding skills, while investing in my hobby of TTRPGs.

The goal of the project was to create a live website that integrated the following:

- Ads (integrated through google adsense)

- User accounts (through firebase)

- Online Marketplace (through stripe)

This is a work in progress, and more is to come soon...
