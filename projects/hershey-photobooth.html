Project Showcase

Hershey Photo Booth Kiosk

A customized photo experience for retail engagement.

Overview

I utilized the open-source Google Photobooth as a foundation to create a custom retail experience. This enhanced application allows users to take or upload photos and customize them with unique designs. The seamless integration allows customers to purchase a Hershey's candy bar wrapped in their own personalized wrapper. The software is deployed on kiosks in the Hershey's Chocolate World store in New York Times Square and several other locations.

Technical Details

Built using Flutter/Dart for cross-platform performance. The application connects to a robust cloud backend to handle high-resolution image processing and secure data storage.

  • Branded Experience: Integrated Hershey's specific assets including mascot characters and themed stickers.
  • Dynamic Content: Supports dynamically updating images and assets based on the specific kiosk location.
  • Customization: Added text editing capabilities allowing users to personalize their wrappers with names and messages.
  • Localization: Implemented multiple language support to accommodate international tourists.
  • Cloud Integration: Utilized AWS services via API for scalable image handling, storage, and processing.
