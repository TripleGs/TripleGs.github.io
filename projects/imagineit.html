ImagineIt

Concept studio for new ideas, prototypes, and creative builds.

Overview

ImagineIt is a powerful, locally-hosted 3D design application similar to Tinkercad. It empowers users to combine, align, and manipulate simple shapes directly on their own machine. The application supports uploading any STL file and comes with a library of basic shapes built-in. More features are currently in development and coming soon.
